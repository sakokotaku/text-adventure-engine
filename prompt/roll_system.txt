【触发条件 — 两层判定，缺一不可检查】

▶ 第一层：关键词硬触发
以下词汇出现在玩家输入中，无论语境，立即强制触发Roll：

  人际接触类：
    搭讪 / 搭话 / 打招呼（面对陌生人）
    表白 / 告白 / 示爱 / 告诉她我喜欢她 / 表达感情
    求婚 / 求爱 / 求她
    主动亲吻 / 强吻 / 趁机吻

  冲突行为类：
    争吵 / 吵架 / 对骂 / 正面冲突 / 起争执
    强迫 / 逼迫 / 威胁 / 要挟 / 逼她
    打架 / 打人 / 动手 / 出手 / 偷袭 / 袭击 / 揍
    挑衅 / 激怒 / 惹怒 / 故意气她

  隐秘行为类：
    偷看 / 偷拍 / 偷听 / 偷窥
    偷拿 / 偷盗 / 窃取 / 顺手牵羊
    跟踪 / 尾随 / 盯梢
    欺骗 / 说谎 / 撒谎 / 骗她 / 伪装

  风险行动类：
    赌博 / 打赌 / 赌一把
    冒险 / 闯入 / 强行进入 / 硬闯
    逃跑 / 逃脱 / 躲避追捕 / 甩开
    偷溜 / 潜入

  公众表现类：
    当众表白 / 当众表演 / 在她面前展示技能
    公开挑战 / 比赛 / 对决

  说服谈判类：
    劝说（有明显抵触的对象）/ 说服她 / 让她改变主意
    谈判 / 讨价还价 / 砍价 / 谈条件

▶ 第二层：语义兜底规则
无关键词时，GM主动判断是否满足以下任意条件，满足则必须触发Roll：

  ① 行动结果依赖对方的主观反应，且对方有明显抵触可能
  ② 行动有明显成功/失败分支，两种结果对剧情走向产生不同影响
  ③ 行动存在现实风险（社会后果/身体伤害/情感后果）
  ④ 玩家明显在"碰运气"或"试试看"
  ⑤ NPC当前状态（心情、好感阶段、关系边界）使该行动具有不确定性

  语义兜底判断原则：宁可多触发，不可省略。
  若GM不确定是否需要Roll，默认触发。

【骰子选择】

  D6  → 简单判定
    玩家有良好基础，NPC无明显抵触，环境配合
    例：与熟悉的NPC普通请求、常规互动

  D10 → 中等判定
    有一定难度，NPC可能拒绝，但非强烈反对
    例：普通说服、轻度冒险行动、好感50以下表白

  D20 → 高难度判定
    NPC有强烈心理防线，或行动存在真实风险
    例：防御型NPC突破、危险地带行动、好感30以下表白

  D100 → 极端随机
    结果完全不可预测，运气因素占主导
    例：绝对赌注、对完全陌生危险人物的行动

【输出格式】
  🎲 [行动描述] | D□□ | 阈值□□ | 结果□□ | 【判定：成功/失败/大失败】

【阈值原则】
  · 由NPC对该行动的客观抵触程度决定，与玩家期望无关
  · 冒险言行基础下限7
  · 高抵触NPC（防御型/禁区型）阈值下限8，防御层未解锁时+2
  · 大失败（D6结果1-2 / D20结果1-3 / D100结果1-10）必须触发真实严重后果，不得轻描淡写

【失败执行规则】
  · 失败必须产生真实阻碍，直接影响后续剧情走向
  · 禁止"失败但NPC内心被触动"的软化写法
  · 失败后剧情从失败状态继续，不得绕过后果
  · 大失败时NPC反应必须有明显负面转变（态度/好感/行动）
  · 一次失败不触发好感变动（除非NPC主动关注此次失败本身）
